# 🍽️ Restaurant Menu Management System

## 📌 Objective
This project is developed using Python and SQLite to manage restaurant menu items efficiently. It demonstrates database integration and CRUD operations.

---

## ⚙️ Features
- Add categories
- Add menu items
- View menu items
- Update item price
- Delete items
- Delete categories
- CLI-based user interface

---

## 🛠️ Technologies Used
- Python
- SQLite3

---

## 🗄️ Database Structure

### Categories Table
- id (Primary Key)
- name (Unique)

### Menu Items Table
- id (Primary Key)
- name
- price
- category_id (Foreign Key)
